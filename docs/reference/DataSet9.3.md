# Data for Example 9.3 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

DataSet9.3 is used for Response surface design with incomplete blocking

## Usage

``` r
data(DataSet9.3)
```

## Format

A `data.frame` with 28 rows and 4 variables.

## Details

- block with 7 blocks

- a is a factor with 3 levels 0,-1 and 1

- b is a factor with 3 levels 0,-1 and 1

- c is a factor with 3 levels 0,-1 and 1

- y is a Gaussian response variable
