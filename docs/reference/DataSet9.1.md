# Data for Example 9.1 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

DataSet9.1 is used for Nested factorial structure

## Usage

``` r
data(DataSet9.1)
```

## Format

A `data.frame` with 30 rows and 4 variables.

## Details

- block 10 blocks

- trt 6 treatments nested within sets

- set 2 sets

- y is a Gaussian response variable
