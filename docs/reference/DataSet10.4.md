# Data for Example 10.4 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

DataSet10.4 is used for demonstrating the relationship between BLUP and
Fixed Effect Estimators

## Usage

``` r
data(DataSet10.4)
```

## Format

A `data.frame` with 32 rows and 3 variables.

## Details

- a is a factor with 2 levels

- b is a factor with 8 levels

- y is a Gaussian response variable

## References

1.  Stroup, W. W., Ptukhina, M., and Garai, S. (2024). *Generalized
    Linear Mixed Models: Modern Concepts, Methods and Applications (2nd
