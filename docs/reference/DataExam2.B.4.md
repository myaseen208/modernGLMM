# Data for Example 2.B.4 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

Exam2.B.4 is used to illustrate one way treatment design with Binomial
observations.

## Usage

``` r
data(DataExam2.B.4)
```

## Format

A `data.frame` with 6 rows and 4 variables.

## Details

- obs number of observations

- trt three treatments with class factor

- Nij number of Bernoulli trials on each individual

- y number of successes on each individual

## References

1.  Stroup, W. W., Ptukhina, M., and Garai, S. (2024).*Generalized
    linear mixed models: modern concepts, methods and applications*. CRC
