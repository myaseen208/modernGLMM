# Data for Table1.2 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

Exam1.2 is used to see types of model effects by plotting regression
data

## Usage

``` r
data(Table1.2)
```

## Format

A `data.frame` with 36 rows and 5 variables.

## Details

- X have 11 levels in varying intervals from 0 to 48 observed for
  multiple batches

- Y continuous variable observed at each level of X

- Fav number of successes

- N number of bernoulli trials

- Batch Batches as 1, 2, 3, 4
