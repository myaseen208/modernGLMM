# Data for Example 2.B.2 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

Exam2.B.2 is used to visualize the effect of glm model statement with
binomial data with logit and probit links.

## Usage

``` r
data(DataExam2.B.2)
```

## Format

A `data.frame` with 11 rows and 3 variables.

## Details

- x independent variable

- n Bernoulli trials (Bernoulli outcomes on each individual)

- y number of successes on each individual
