# Data for Example 2.B.7 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

Exam2.B.7 is related to multi batch regression data assuming different
forms of linear models with factorial experiment.

## Usage

``` r
data(DataExam2.B.7)
```

## Format

A `data.frame` with 16 rows and 4 variables.

## Details

- Rep number of replications

- a factor with two levels 1 and 2

- b factor with two levels 1 and 2

- y response variable

    linear mixed models: modern concepts, methods and applications*. CRC
