# Data for Example 10.1 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

DataSet10.1 is used for One-way random effects model and BLUP estimation

## Usage

``` r
data(DataSet10.1)
```

## Format

A `data.frame` with 24 rows and 2 variables.

## Details

- a is a factor with 12 levels

- y is a Gaussian response variable
