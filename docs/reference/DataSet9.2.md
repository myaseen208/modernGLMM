# Data for Example 9.2 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

DataSet9.2 is used for Incomplete strip-plot ( 3 cross 3 factorial).

## Usage

``` r
data(DataSet9.2)
```

## Format

A `data.frame` with 36 rows and 6 variables.

## Details

- block 9 blocks each consisting of 2 rows and 2 columns

- a is a factor with 3 levels assigned at random to rows

- b is a factor with 3 levels assigned at random to columns

- y is a Gaussian response variable
