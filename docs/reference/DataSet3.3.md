# Data for Example3.7 from Generalized Linear Mixed Models: Modern Concepts, Methods and Applications by Stroup, Ptukhina, and Garai (2024, 2nd ed.)

DataSet3.3 is used to see types of model effects by plotting regression
data

## Usage

``` r
data(DataSet3.3)
```

## Format

A `data.frame` with 36 rows and 6 variables.

## Details

- X Each batch observed at several times:0,3,6,12,24,36,48 months

- Y continuous variable observed at each level of X

- Fav number of successes

- N independent Bernoulli trials

- Batch Batches as 1, 2, 3, 4

- Count binomial response variable
